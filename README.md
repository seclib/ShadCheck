# 🎮 ShadCheck

ShadCheck est une application GUI permettant de vérifier la compatibilité des jeux PS4 avec shadPS4.

## Features
- 🔎 Recherche (Nom / CUSA)
- 🎮 Statut compatibilité
- 📂 Scan local de jeux
- ⚡ Cache rapide
- 🎨 UI moderne (dark mode + cards)

## Installation
pip install -r requirements.txt

## Lancement
python main.py
